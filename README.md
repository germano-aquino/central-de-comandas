# Central-de-Comandas

Este repositório tem por objetivo criar uma aplicação em javascript que substitua uma comanda em papel de uma clinica de estética. Esta aplicação deve cumprir o seguinte:

- Anotar os Serviços à serem executados
- Realizar uma pesquisa de preferências do Cliente
- Ser assinado por rubrica com o mouse o touch
- Enviar as informações coletadas por PDF
