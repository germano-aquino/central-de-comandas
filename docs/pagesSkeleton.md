# Páginas da central de comandas

## Página de primeiro acesso

- Criar usuário administrador
- Criar unidade de atendimento
- Criar categoria (opicional)
- Criar serviço
- Criar seções do Questionário (opicional)
- Criar pergunta do Questionário

## Página de Login

- Campo de login com email e senha
- Hyperlink com esqueci a senha

## Página de Cadastro

- Cadastro de Unidades
- Cadastro de Usuários
  - Administrador
  - Recepcionista
  - Prestador de Serviço
- Cadastro de Categoria de Serviços
- Cadastro de Serviços
- Cadastro de Seções de Questionários
- Cadastro de Perguntas

## Página de Comanda

- Dados do cliente (nome e numero de cell)
- Separação de Serviços por Categorias
- Carrinho com todos serviços e valores
- Forma de Pagamento

## Página de Questionário

- Perguntas separadas por Seções
- Assinatura do cliente (touch/mouse)
- Finalizar atendimento

## Página de Atendimentos

- Listar os últimos atendimentos criados
- Filtrar os atendimentos por:
  - Unidade
  - Profissional
  - Cliente
  - Categoria de Serviço
  - Serviço
  - Estado do Atendimento (Aberto/ Em andamento/ Finalizado)
  - Data/Hora do Atendimento

## Página do Atendimento

- Dados do cliente
- Dados do(s) prestadore(s) de serviço(s)
- Serviços realizados
- Perguntas e respostas do questionário
- Assinatura do Cliente
- Gerar PDF para impressão
- Alterar estado do atendimento
